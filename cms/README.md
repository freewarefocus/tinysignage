# CMS Frontend

Vue 3 + PrimeVue single-page app. The build output (`../app/static/cms/`) is committed to the repo so that installs don't require Node.js.

## Development

```bash
npm install
npm run dev       # dev server with hot reload
npm run build     # production build → ../app/static/cms/
```

After making changes, run `npm run build` and commit the updated `app/static/cms/` directory.
